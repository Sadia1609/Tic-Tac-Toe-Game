# Tic-Tac-Toe

A simple two-player Tic-Tac-Toe game built with HTML, CSS, and JavaScript.
live: https://tic-tac-toe-game-git-main-sadia-rahmans-projects-aa9a64e0.vercel.app/

## How to Play

- Player O starts first
- Click any box to place your mark
- First to get 3 in a row (horizontal, vertical, or diagonal) wins

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Features

- Alternating turns between X and O
- Winner detection
- Disables boxes after click to prevent overwriting

## What I Practiced

- DOM manipulation
- Event listeners
- Arrays and loops for win pattern checking
- Game state management
